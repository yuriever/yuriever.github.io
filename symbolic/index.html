Paclets and packages

  • Package dependency

  • lily`arxiv`

    • A Mathematica package for downloading preprints and generating BibTeX by arXiv ID.

  • lily`class`

    • This package is designed to manage a specific form of class composition.

  • lily`algebra`

    • A Mathematica package for implementing and managing finitely presented associative algebras.

  • lily`preindex`

    • A Mathematica package for coding a indexed variable into a single symbol.

  • lily`index`

    • A Mathematica package for coding/decoding a variable indexed by label and script into/from a single symbol.

  • lily`cartesian` and lily`cartesianIndex`

    • Mathematica packages handling Cartesian coordinates and metrics.

Wolfram technology conference

  • Inside the Mathematica Pattern Matcher

    • This talk gives an overview of how the pattern matcher work, and discusses the relationship between the two major pattern matching paradigms in Mathematica: matching on general expressions and on strings.

  • Working with unevaluated expressions

    • This talk demonstrates situations in which evaluation control is important, points out common pitfalls and provides useful tools and techniques.

  • How to Use Inactive

    • Inactive prevents specific functions from firing without suppressing all evaluation. This finer-grained control makes it easy to build up partially evaluated expressions programmatically.
