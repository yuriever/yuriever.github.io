Package development and dependency

References:

  • Paclets and paclet development tutorial/Paclets

扩展包结构可概括为：

p.wl
(*用Get强制重新读取扩展包p0.wl...*)
Get/@{"p0`",...}

(*设定$Context为p`，Needs扩展包p1.wl...*)
BeginPackage["p`",{"p1`",...}];

(*重载扩展包时清除上次的定义*)
Unprotect@@Names["p`*"];
ClearAll@@Names["p`*"];

(*声明公共函数*)
foo::usage = "";

(*设定$Context为p`Private`*)
Begin["`Private`"];

(*函数定义*)
foo = ...;

(*结束当前$Context，恢复为p`*)
End[];

(*保护所有未指名的符号*)
protectAllExcept[
    "foo1"
];

(*恢复$Context，并添加p`至$ContextPath*)
EndPackage[];

Function type

  • 常数 (constant)，存储变量的列表以及列表操作函数；

  • 对象 (object)，存储对象属性的关联，以及实现对象方法的函数；

  • 判别函数 (questing function)，用以界定变量的类型和范围，因此定义尽量完备；

  • 主体函数 (body function)，实现该扩展包的各种功能。为防止意外计算，需要定义再去定义，遵循自上到下的书写规则；

  • 打印函数 (format)，区分返回值和打印，分离函数之间传递的值和输出结果。

Naming

  • 同一类功能的函数以相同的前缀命名。

  • 判别函数的命名后缀为 Q。

  • 全局变量 foo 的局部拷贝命名为 $$foo，重复模式的名称为单词复数 foos__，列表模式的名称为 fooList_，helper 函数命名为 foo`foo1。
