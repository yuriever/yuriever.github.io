    \section{Distributions on \(l\)-space}

    The Cantor set \(C\) and the 2-adic integers are both identified as the space of functions from \(\N\) to \(\set{0,1}\) with the totally disconnected topology. Smooth functions on the Cantor set are defined as locally constant functions. Regarding \(\N\) as the base space and \(\set{0,1}\) as the target space, the integration region of path integral is the Cantor set.

    The path integral is \begin{equation} Z=\sum_{x\in C}f(x) \end{equation} for some Boltzmann weight \(f(x)\).

    The topology of Cantor set does bookkeep the poset structure of \(\N\). This should reflect the locality on the \(p\)-adic integers.
