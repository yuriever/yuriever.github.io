    0. 背景¶

    本文考虑的标量场的作用量为

    \[ S=\frac{1}{2}\int\,{d^{d}x}\sqrt{-g}\,( \partial^{a}\phi \partial_{a}\phi + c_s R \phi^2) \]

    选取合适的系数 \(c_{s}=\frac{d-2}{4(d-1)}\) 后，修正项 \(R \phi^2\) 可使作用量在 Weyl 变换下不变，因此能动张量 \(T_{ab}=\frac{2}{\sqrt{-g}} \frac{\delta S}{\delta g^{ab}}\) 的迹为零 \(\trace{T}=0\)，该理论称为共形标量场 (conformal scalar)。

    电磁场的作用量为

    \[ S=\frac{1}{4}\int\,{d^{d}x}\sqrt{-g}\,F_{ab}F^{ab} \]

    且有 \(\trace{T}= -\frac{d-4}{4}F^{2}\)。事实上，仅当 \(d=4\) 时，自由电磁场具有共形对称性。

    1. 导言¶

    变分计算问题回到最底层，是遵循一系列规则依照一定次序化简表达式，逐步降低表达式的复杂度，最后抵达目标表达式。

    这是一个代数问题，不需要指明场构型在哪一个函数空间中，不需要流形的整体信息，不必明确是何种意义的积分1。我们所需的只是一个坐标卡内的局部结构，例如微分、度规、复结构。这个坐标卡可以是欧氏空间的开集，可以是超空间的开集，或者来自于某种形式几何。

    say some words on variational bi-complex, formal geometry and BV-BRST.

    那么直接的实现方案是利用MMA的核心功能：Rule 和 Replace，相关介绍可见符号计算：非交换代数。如果想重复利用代码，需要明确各个对象/变量之间的派生关系，例如：某张量场是在某坐标卡上的，某联络的某一项是关于某纤维丛的。xAct 提供了相应的完整平台。

    我们需要调用的扩展包如下：

    2. 流形与作用量¶

    我们先声明流形、度规和 Levi-Civita 联络，号差为\((-,+,+,...)\)。函数 DefManifold 可声明流形，其三个参数分别为流形的名称、维数与指标集合。注意 xAct 会自动声明切丛以及相关的张量丛。类似的，声明度规的函数为 DefMetric。

    (*dim为xAct的预定常量，打印为d*)
(*函数IndexRange指定了所有英文字母为抽象指标*)
DefManifold[M, dim, IndexRange[a, z]];

(*第一个变量是度规行列式的符号*)
(*下指标用负号标记，xAct自动定义适配的Levi-Civita联络，SymbolOfCovD和PrintAs是打印规则*)
DefMetric[-1, metric[-a, -b], CD, SymbolOfCovD -> {";", "\[Del]"}，PrintAs -> "g"]

measure[met_?MetricQ] := Sqrt[-ToExpression["Det" <> ToString@met <> "[]"]];
    方便起见，这里引入函数 measure，measure[metric] 为 \(\sqrt{-g}\)。在声明完度规 metric 后，xAct 会自动声明 Detmetric[] 为附着 metric 的度规行列式。

    声明常量 \(c_{s}\)、标量场 \(\phi\)、电磁场 \(A_{a}\) 与 \(F_{ab}\)， 

    DefConstantSymbol[cs, PrintAs -> "c"]
DefTensor[\[Phi][], M4]
DefTensor[A[-a], M4]
DefTensor[F[-a, -b], M4, Antisymmetric[{-a, -b}]]

    场强和规范势需满足规则 \(F_{ab}=\partial_{[a}A_{b]}\) ，xAct 提供了类似 Rule 的函数 MakeRule，以及类似 SetDelayed 的函数 IndexSetDelayed，

    ?MakeRule
?IndexSetDelayed

    MakeRule[{LHS, RHS}] gives a list of delayed rules with left and right hand sides as given by LHS and RHS respectively, but moving indices to consider all possible equivalent rules. See notes for its options PatternIndices, MetricOn, UseSymmetries and TestIndices. MakeRule[{LHS, RHS, conditions}] adds conditions to the rules.

    IndexSet[lhs, rhs] defines a relation between indexed expressions lhs and rhs, without evaluating rhs.

    这里我们选用 MakeRule，更方便控制计算。注意与 Rule 不同，MakeRule 不需要使用模式。场强转化为规范势的规则为

    ruleFToA = MakeRule[{F[a, b], CD[a]@A[b] - CD[b]@A[a]}];

    接下来定义 Lagrangian density，以下混称作用量，

    action[\[Phi]] = 1/2 (CD[a]@\[Phi][] CD[-a]@\[Phi][] + cs RicciScalarCD[] \[Phi][]^2 ) measure[metric];
action[A] = 1/4 F[-a, -b] F[a, b] measure[metric];

    如此便完成了准备工作。

    3. 能动张量¶

    扩展包 xPert 提供了变分与度规微扰的相关函数。变分函数 Perturbation 内置了变分满足的线性性和 Lebniz 规则，例如 Perturbation@A[-a] 便是 \(\delta A_{a}\)，被打印为 \(\Delta A_{a}\)，例如对电磁场作用量的变分为

    action@A // Perturbation

    对作用量中的三个因子的分别变分，展开给出三项，

    被变分的基本场为规范势 \(A\) 而非场强 \(F\)，因此利用规则 ruleFToA 展开 \(F\)，

    actionEM /. ruleFToA // tensorSim // Perturbation

    注意 \(\delta A^{a}\) 隐含度规缩并 \(\delta(A_{b}g^{ab})=\delta A_{b} g^{ab}+A_{b}\delta g^{ab}\)。需要将 \(\delta L\) 展开成 \((\dots )\delta g+(\dots )\delta A_{a}\) 的形式，\(\delta A_{a}\) 的系数给出场方程，\(\delta g\) 的系数给出能动张量，括号中含有关于变分的协变导数，需要通过分部积分来消除。

    函数 VarD[#1,#2] 提供了抽取系数以及分部积分的功能，其中第一个自变量表示变分场，第二个自变量表示关于协变导数 CD 进行分部积分。此外，在声明度规时已自动定义了度规变分，由此可计算能动张量以及迹，

    emtensor[field_] := 2/measure[metric] VarD[metric[a, b], CD]@action[field] // tensorSim;
traceT[field_] := emtensor[field] metric[a, b] // tensorSim;

    结果为 

    emtensor@A
traceT@A
traceT@\[Phi]

    注意这里无需使用规则 ruleFToA，如果提前展开场强为规范势，能动张量的表达式会略显复杂，稍后介绍化简方法。

    对于电磁场，仅当 \(d=4\) 时 \(\trace T=0\)。对于标量场，取待定系数 \(c_{s}=\frac{d-2}{4(d-1)}\)，\(\trace T\) 为

    4. 运动方程¶

    在 xPert 的基础上，xTras 进一步封装了关于变分的规则和函数，此时函数 VarD 的第一个变量可以接收其它张量场，由此可计算运动方程，

    eom[field_, rule_ : {}] := 1/measure[metric] VarD[indexed[field], CD]@(action[field] /. rule) // tensorSim;

    此处函数 indexed 的作用为赋予已声明的张量场以默认的指标，例如

    indexed /@ {A, F, \[Phi], B}

    给出

    标量场与电磁场的运动方程为

    eom@\[Phi]
eom[A, ruleFToA]

    由此可验证共形标量场的能动张量迹为零。
